What happens behind the scenes:
  1. Form validates all fields (email regex check)
  2. POST /api/track-enquiry fires in background
  3. Booking code generated (TND-XXXXX)
  4. Reservation saved to database (status: enquiry)
  5. UTM / referrer attribution captured
  6. Customer confirmation email sent
  7. Admin notification email sent
  8. WhatsApp opens with pre-filled message

Waitlist automation chain:
  1. Captain cancels a confirmed booking
  2. System checks waitlist for that sailing date
  3. Auto-creates enquiry reservations for each waitlisted customer
  4. Sends "spot opened up" email to each customer
  5. Marks waitlist entries as notified
  6. Captain sees new enquiries to confirm
